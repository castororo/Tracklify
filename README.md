# Tracklify

Project Management & Analytics Dashboard.

## Project Structure
- **/client**: React Frontend (Vite + Tailwind + Shadcn)
- **/server**: Node/Express Backend (MongoDB)

## Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB

### Installation
1.  **Install Dependencies**:
    ```bash
    npm run install:all
    ```

### Running the App
1.  **Start Backend**:
    ```bash
    npm run dev:server
    ```
2.  **Start Frontend**:
    ```bash
    npm run dev:client
    ```

## Features
-   Authentication (JWT + Google OAuth)
-   Role-based Access Control
-   Real-time Dashboard Analytics
-   Project & Task Management
-   Activity Logging
